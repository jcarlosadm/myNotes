#Chapter 1 - Start to Code

Saída padrão:

```python
print("message")
```

Entrada padrão:

```python
myVar = input("message")
myIntVar = int(myVar)
```

Decisão **if/else**:

```python
if guess == 5:
    print("you win!")
else:
    print("you lose!")
print("Game over!")
```

Loop **while**:

```python
while answer == "no":
    answer = input("Are we there?")
print("We're there!")
```

Para comentários use **#** para comentar uma linha ou **'''** para comentar várias linhas.
